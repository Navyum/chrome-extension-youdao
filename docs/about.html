关于有道云笔记导出助手

有道云笔记导出助手（YoudaoNote Export）是一个开源 Chrome 扩展，帮助有道云笔记用户一键批量导出笔记为本地 Markdown。我们相信：你的笔记数据应该可以被你随时带走。

我们的理念

云笔记工具适合长期积累，但迁移、归档和灾备不能被复杂流程卡住。这个项目的目标很明确：让普通用户不需要命令行、不需要手动复制 Cookie，也能把自己的有道云笔记保存为通用、可读、可迁移的 Markdown 文件。

项目完全开源，核心处理均在浏览器本地完成。扩展只复用你在浏览器中的 note.youdao.com 登录态读取笔记数据，不运行保存你笔记内容的中转服务。

核心能力

  • 批量导出选中的笔记本和文件夹。
  • 导出 Markdown 文件，并把图片和白板等依赖资源保存到同级 assets 目录。
  • 转换表格、思维导图、Draw.io、Excalidraw 和图片文件等常见有道特殊格式。
  • 后台执行导出任务，关闭弹窗不影响进度。
  • 导出完成后自动打包为 ZIP 下载。
