隐私政策

最后更新日期: 2026年7月15日

欢迎使用 有道云笔记导出助手。本隐私政策说明扩展在导出有道云笔记时如何处理数据，以及我们如何保护你的隐私。

1. 核心隐私原则

我们相信你的笔记内容属于你。扩展采用本地优先的处理方式：笔记读取、格式转换、资源整理和 ZIP 打包都在你的浏览器中完成。我们不运行用于接收、保存或分析你笔记内容的服务器。

2. 扩展会访问哪些数据

2.1 浏览器登录状态

扩展会使用 Chrome 的 cookies 权限读取有道云笔记相关 Cookie，用于确认你已经登录 note.youdao.com，并以你的身份调用有道云笔记接口。Cookie 不会被上传到第三方服务。

2.2 笔记与资源数据

当你主动发起导出时，扩展会读取你选中的笔记本、文件夹、笔记正文和引用资源，用于生成 Markdown、assets 目录和最终 ZIP 文件。这些数据仅在本地处理。

2.3 本地任务状态

扩展会使用浏览器本地存储保存导出进度、任务状态和必要的临时信息，以便关闭弹窗后继续展示进度或恢复界面。

3. 扩展不会做什么

  • 不会上传你的笔记内容到任何第三方服务器。
  • 不会收集或出售个人信息。
  • 不会修改或删除你在有道云笔记中的原始数据。
  • 不会绕过有道云笔记的权限控制；只能导出当前登录账号有权访问的内容。

4. 权限说明

  • cookies：读取有道云笔记登录状态，用于自动鉴权。
  • storage：保存本地任务状态、进度和必要配置。
  • downloads：下载生成的 ZIP 文件。
  • youdao.com 主机权限：访问有道云笔记接口和资源地址，完成你主动发起的导出任务。

5. 第三方服务

扩展导出功能只与有道云笔记相关域名交互，用于读取你已登录账号可访问的内容。站点本身不包含第三方追踪脚本或分析代码。

6. 你的控制权

你可以随时停止导出、清理扩展本地任务状态，或通过卸载扩展删除其本地存储数据。你也可以在 Chrome 扩展管理页调整站点访问权限。

7. 联系我们

如果你对本政策有任何疑问，请通过 邮件 联系我们，或在 GitHub 提交问题。
