# Chrome 应用商店描述

## 简短描述

将有道云笔记导出为本地 Markdown 备份，支持图片、文件夹和常见特殊格式。

## 完整描述

有道云笔记导出可以帮助你在浏览器中备份和迁移有道云笔记。它会加载你的笔记本，将选中的文件夹导出为 Markdown，下载引用的图片，并将所有内容打包成 ZIP 文件。

扩展会保留笔记本结构，并将 Markdown 依赖资源保存到本地 `assets` 文件夹。它还支持处理常见有道格式，例如表格、思维导图、Draw.io 图表、Excalidraw 白板和图片文件。

主要功能：

- 批量导出有道云笔记和笔记本
- 将笔记转换为 Markdown
- 保留文件夹结构
- 本地化图片和链接资源
- 将表格和思维导图导出为 Markdown
- 将 Draw.io 和 Excalidraw 导出为 Markdown，并生成 SVG 资源
- 关闭弹窗后仍可在后台继续导出
- 中文和英文界面

隐私：

你的笔记数据会在浏览器扩展本地处理。扩展使用你现有的有道云笔记浏览器登录状态，不会将笔记内容上传到任何外部服务。
