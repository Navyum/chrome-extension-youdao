一键批量导出

自动加载笔记本树，选择要导出的笔记本或文件夹后即可批量处理，无需逐篇复制粘贴。

保留笔记本结构

导出的 Markdown 会按有道云笔记中的笔记本和文件夹层级组织，迁移后仍能保持原有知识结构。

Markdown 与资源本地化

笔记统一转换为 Markdown，图片和白板 SVG 等依赖资源保存到同级 assets 目录，链接使用本地相对路径。

特殊格式转换

表格转 Markdown 表格，思维导图转大纲，Draw.io 和 Excalidraw 转为 Markdown 并链接本地 SVG。

后台任务不中断

导出任务在扩展后台继续执行，关闭弹窗不会中断。重新打开后可继续查看导出进度。

隐私安全保障

所有转换和打包都在你的浏览器本地完成。扩展不会将笔记内容上传到任何第三方服务。

我们重视你的隐私

有道云笔记导出助手只作为你和 note.youdao.com 之间的本地导出工具。扩展使用浏览器登录态读取你有权访问的笔记，并在本地完成转换、资源下载和 ZIP 打包，不运行保存笔记内容的中转服务器。

常见问题

关于有道云笔记导出助手的疑问解答

有道云笔记导出助手是免费的吗？
是。项目免费使用，并以开源方式维护。你可以用它做个人笔记备份、迁移和长期归档。
支持哪些导出格式？
当前导出结果为 Markdown 文件和同级 assets 资源目录。图片、Draw.io、Excalidraw 等资源会保存到 assets；表格和思维导图会转换为 Markdown 内容。
需要手动复制 Cookie 或 Token 吗？
不需要。只要你已在浏览器中登录 note.youdao.com，扩展会读取浏览器登录状态完成鉴权。
导出会影响有道云笔记里的原始数据吗？
不会。工具只读取数据并在本地转换格式，不会修改、删除或上传你在有道云笔记中的内容。
关闭弹窗会中断导出吗？
不会。导出任务会在扩展后台执行，关闭弹窗后仍会继续。重新打开弹窗可以查看当前进度。
数据安全有保障吗？
所有核心处理都在本地浏览器内完成。扩展不会把笔记内容发送到第三方服务器，也不收集用户数据。
