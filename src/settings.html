通用设置

当前版本无需额外配置，保持浏览器登录即可导出。

鉴权说明：只要保持在浏览器中登录 note.youdao.com，扩展会自动读取 Cookie 完成鉴权，无需手动配置 Token。

导出行为

导出结果统一为 Markdown 文件和同层 assets 资源目录。普通图片、Draw.io、Excalidraw 等资源会放入 assets；mindmap 和表格会转换为 Markdown。

导出任务会在后台执行，关闭弹窗不会中断。完成后浏览器会自动下载 ZIP。

关于 YoudaoNote Export

零门槛、一键式的有道云笔记批量导出工具

项目说明

YoudaoNote Export 是一款专为有道云笔记用户打造的 Chrome 扩展，帮助你一键批量导出笔记为 Markdown 格式。无论是迁移到 Obsidian、Notion，还是做定期备份，都能轻松搞定。

所有数据处理完全在本地完成，不收集、不上传任何用户数据。

功能特性

批量导出一键导出所有笔记本，保留完整目录结构
Markdown 转换XML/JSON 格式笔记自动转换为标准 Markdown
图片本地化自动下载笔记中的图片，替换为本地相对路径
ZIP 打包导出结果自动打包为 ZIP 文件，一次性下载
实时进度可视化进度条，支持暂停和继续导出
隐私安全纯本地处理，零数据上传，自动读取浏览器 Cookie
